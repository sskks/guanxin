### 已知问题与技术债

#### 高优先级

**单文件巨型化**：index.html 达 5364 行，所有 CSS/HTML/JS 挤在一个文件中，导航困难、协作易冲突、无法测试。建议拆分为 index.html + style.css + app.js + data.js 四文件（保持纯静态，不引入构建工具）。

**易学数据硬编码**：约 900 行的卦辞、爻辞、象辞等数据直接写在 JS 中，任何勘误都要改主文件。建议提取为 `data/hexagrams.json` 独立文件。

**无自动化测试**：八字计算、梅花易数算法、八宅风水计算均无测试验证正确性。建议至少为纯计算函数添加单元测试。

#### 中优先级

**API Key 安全**：localStorage 中明文存储 API Key，无 CSP 头保护。建议添加 Content-Security-Policy meta 标签限制脚本来源。

**无障碍性缺失**：无 ARIA 标签、无键盘导航、无屏幕阅读器支持、部分信息仅通过颜色区分。

**性能优化空间**：64卦数据（900行 JS）每次访问全量加载；八宅罗盘每次重新生成 200 行 HTML；分享卡片 Canvas 生成阻塞主线程。

**浏览器兼容**：使用了 IndexedDB、DeviceMotion、Web Share、CSS Custom Properties 等现代 API，无 polyfill 回退。

#### 低优先级

**Python 补丁脚本清理**：7 个 .py 补丁文件（alm3.py 等）已无运行时用途，仅为开发历史参考，可考虑移入 `patches/` 子目录或归档。

**离线字体回退**：Google Fonts 通过 CDN 加载，离线时无回退字体。

**IndexedDB 版本管理**：当前版本为 1，无数据迁移路径，schema 变更可能导致用户数据丢失。
