## 观心（Guān Xīn）产品需求文档

> 版本：v1.0 MVP | 最后更新：2026-06-22

---

### 产品定位

观心是一款基于周易文化的单页 Web 应用，融合梅花易数占卦、命宫解读、六十四卦图鉴等功能，以古朴水墨风格呈现传统易学智慧。面向对易经文化感兴趣的普通用户，提供轻量级的日常占卦与学习体验。

### 目标用户与场景

对周易、传统文化有兴趣的年轻用户（18-40岁），日常在手机上使用，核心场景包括：每日品读一卦获取生活指引、遇事不决时通过占卦求问、系统学习六十四卦知识、了解个人命宫与流年运势。

### 竞品参考

市面上周易类 App 多为重功能型（排盘工具、命理分析），界面老旧且广告密集。观心走差异化路线——轻量美学 + AI 解读 + 单页即达，不注册、不付费、不臃肿。

---

### 功能清单

#### P0 - 核心功能

**每日一卦**：根据当前日期自动展示对应卦象，包含卦图、卦辞、象辞，支持前后日翻页。用户可以每日品读一卦，养成学习习惯。验收标准：打开页面自动显示当日卦象；左右滑动可切换日期；卦辞和象辞完整展示。

**竹签摇卦**：点击摇卦按钮或物理摇晃手机，播放竹签动画后随机生成卦象，以 3D 翻牌方式揭晓结果。验收标准：动画流畅无卡顿；支持 DeviceMotion 物理摇晃触发（移动端）；结果卦象信息完整。

**问易（以事问卦）**：用户输入心中所惑的问题，系统以梅花易数算法基于毫秒级时间戳生成卦象，并可选 AI 解读。验收标准：输入问题后可生成卦象；结果包含卦图、卦辞、变爻；AI 解读格式化为"行动卡"样式。

**数字起卦**：用户输入两个数字，上卦=数1%8、下卦=数2%8、变爻=(数1+数2)%6，生成卦象。验收标准：两个数字输入框可用；计算逻辑正确；结果展示完整。

**占卦结果页**：统一的结果展示页，包含卦图（CSS 绘制阴阳爻）、卦名、卦辞、象辞、变爻高亮、AI 解读卡片、分享工具栏（复制文字/原生分享/保存图片）。验收标准：所有占卦方式共享此结果页；AI 解读以结构化"行动卡"格式渲染；分享功能三路径均可用。

#### P1 - 重要功能

**命宫解读**：基于用户出生信息（年/月/日/时辰），计算八字、本命卦、命宫、流年，在首页以卡片形式展示。验收标准：生日录入表单完整（年月日+十二时辰）；八字计算逻辑正确；命宫描述和流年分析展示完整。

**八宅风水**：根据用户本命卦计算八宅布局，以罗盘式界面展示八方位对应的游星（生气、天医、延年、伏位、祸害、六煞、五鬼、绝命）及五行关系。验收标准：罗盘布局方位正确；吉凶游星颜色区分清晰；五行生克关系解释完整。

**图鉴百科**：以网格形式展示全部六十四卦，支持按上卦/下卦筛选，点击可查看卦辞、象辞和六条爻辞。验收标准：64卦全部展示无遗漏；筛选功能正确过滤；详情弹窗信息完整。

**占卦历史**：列表展示所有过往占卦记录，支持主题筛选（事业/感情/成长/情绪/其他）、收藏标记、月度报告生成。验收标准：记录按时间倒序排列；收藏可切换；主题筛选正确过滤；月度报告包含统计数据。

**七日回顾**：占卦满7天后，在首页展示回顾卡片，用户可查看原始问题和卦象，标记"已验证"。验收标准：7天后正确弹出回顾记录；标记后不再重复展示；reviewedAt 字段正确写入 IndexedDB。

#### P2 - 辅助功能

**每日签到**：首页签到按钮，连续签到天数以徽章展示。验收标准：每日限签到一次；连续天数计算正确；数据持久化。

**AI 解读**：将卦象数据 + 用户问题发送至通义千问（qwen-plus），返回结构化解读。支持两种调用方式：服务端代理（server.py）和用户自填 API Key 直连。验收标准：解读内容格式化为行动卡；错误时展示友好提示；两种调用路径均可用。

**好友分享**：生成包含占卦数据的分享链接，接收方打开后可查看只读结果。验收标准：链接可正常打开；接收方看到完整卦象信息；无需登录即可查看。

**新手引导**：首次打开时展示多步骤教程覆盖层，介绍核心功能。验收标准：仅首次展示；可跳过；步骤指示器正确。

**设置**：允许用户输入个人 DashScope API Key，用于浏览器直连 AI 解读。验收标准：Key 保存后下次调用生效；提供使用说明。

### 核心用户流程

**流程一：首次使用**
启动页 → 生日录入 → 加载动画 → 新手引导 → 首页（展示八字摘要、本命卦、每日一卦、功能导航卡片）

**流程二：以事问卦**
首页 → 点击"问易" → 输入问题 → 竹签摇卦动画 → 3D 翻牌揭晓 → 占卦结果页 → 可选 AI 解读 → 分享或返回首页

**流程三：每日学习**
首页 → 点击每日一卦 → 查看卦象信息 → 签到打卡 → 返回首页

**流程四：回顾历史**
首页 → 点击历史 → 筛选/收藏 → 查看月度报告 → 点击记录进入结果页

### 视觉风格

整体风格为中国古风水墨画：纸质纹理背景、墨色主调、书法字体标题、土色系配色。设计系统基于 CSS 自定义属性，响应式适配（768px / 480px 断点）。动画风格克制优雅——水墨晕染、竹签轻摇、翻牌翻转。

---

### 技术约束

纯前端单文件 SPA，无框架依赖，无构建工具，无后端服务（AI 解读除外）。数据持久化使用 IndexedDB（占卦记录）和 localStorage（用户设置）。部署于 Cloudflare Pages / GitHub Pages，支持纯静态托管。
