# 观心传统文化与术数内容体系

> 版本：v0.6
> 更新日期：2026-06-24
> 目的：统一项目中《周易》、梅花易数、命宫、八宅、五行、节气等内容的使用边界，避免内容越做越玄、越做越散，同时为后续个性化体验和文化学习功能提供依据。

---

## 1. 总判断

观心当前不应继续定位成“算命工具”，而应定位为：

**以《周易》象数为入口，以情绪整理、生活实践和传统文化学习为落点的东方自我观察产品。**

当前术数体系已经比较多，下一步不是盲目增加更多门类，而是建立一套“准确、克制、可解释、可实践”的内容系统。

核心问题：

- `概念混用`：命宫、命卦、本命卦、八宅命卦之间容易混淆，普通用户看不懂，懂一点传统文化的用户也可能觉得不严谨。
- `内容层级不清`：卦辞、象辞、爻辞、命宫、八宅八星、五行关系都在出现，但缺少统一的“先白话、再术语、再出处、再行动”的解释标准。
- `个性化不够深`：当前个性化主要来自生日、命宫、生日本命卦和流年，但还没有形成“为什么触动我、我卡在哪里、今天怎么练、之后怎么回看”的稳定闭环。
- `准确性边界不足`：代码和文档中仍出现“八字 / 排四柱”字样，但当前并未实现完整四柱八字系统，需要先修正口径。

### 1.0 术数主链路确认

本轮正式确认观心的内容主链路为：

**八卦为体，梅花为用，命宫 / 生日本命卦为人，五行为翻译层，节气为留存层。**

这条链路的含义是：

- `八卦为体`：项目最底层的认知骨架仍是八卦、六十四卦、卦象关系与《周易》象义。
- `梅花为用`：具体问事、每日一卦、问题推演优先使用梅花易数式的体用、互卦、变卦思路。
- `命宫 / 生日本命卦为人`：用于建立“这和我有关”的个体层，承担性格底色、默认反应、长期课题的解释任务。
- `五行为翻译层`：五行负责把术数关系翻译成现代用户能理解的节奏、耗能、压力、滋养与推进语言。
- `节气为留存层`：日更内容、回访理由、轻学习与修身提醒，优先由节气和每日卦象承接。

同时明确一条边界：

**八星八宅可以保留，但不再作为主术数底层，只作为副层观察模块。**

### 1.1 本轮功能方向确认

结合用户反馈和当前工程状态，下一阶段功能方向确认如下：

**主方向：四维生活解析。**

围绕现代用户最关心的四个问题域组织内容，而不是继续堆更多术数门类：

- `事业`：工作压力、行动节奏、目标推进、选择与边界。
- `感情`：亲密关系、沟通误会、关系修复、情绪投射。
- `身心`：身体感受、情绪消耗、睡眠压力、自我照料。此处只做生活提醒，不做医疗判断。
- `钱财`：消费节奏、资源管理、风险意识、长期积累。此处只做财务习惯提醒，不做投资建议。

**术数底座：以周易八卦与梅花易数思路为主，以命宫 / 生日本命卦为个体层，以五行做翻译层。**

短期不新增紫微斗数排盘，也不新增八字十神排盘。原因：

- 紫微斗数虽然有十二宫和星曜结构，适合做人生命题拆分，但完整排盘需要农历年月日时、命身十二宫、十四主星、辅曜、四化、闰月、晚子时等处理，当前单文件前端阶段不适合直接承诺准确排盘。
- 八字十神虽然适合分析性格、关系、事业、财星和官杀等主题，但必须先有完整四柱、日主、十神、旺衰、格局和大运流年。当前项目已经明确不承诺完整四柱八字，因此不应直接加入十神解释。
- 如果现在强行加入，会让产品从“传统文化启发”滑向“专业命理断语”，既增加准确性风险，也会加重用户理解负担。

**预留方向：宫位化表达，而非紫微化排盘。**

可以借鉴紫微斗数“十二宫把人生问题分区”的产品结构，但不使用紫微术语冒充排盘。第一版只做四维：

- 事业宫感：用户在工作和目标上的发力方式。
- 感情宫感：用户在关系里的靠近、表达和边界。
- 身心宫感：用户在压力下的身体和情绪耗能。
- 钱财宫感：用户对资源、安全感和取舍的处理方式。

页面文案应明确这是“观心四维解析”，不是紫微斗数命盘，也不是八字十神排盘。

### 1.2 四维解析的表达边界

四维解析会触及用户真实在意、也最容易脆弱的部分，因此内容必须遵守三条底线：

- `客观但不冷漠`：可以指出风险、卡点和反复模式，但不能用“你一定会”“你注定”“必有灾祸”等断言。
- `不恐吓也不粉饰`：不能制造恐慌，也不能为了安慰而回避问题。表达应是“这里需要留意什么、可以先做什么、什么时候该找现实支持”。
- `术数只是镜子`：所有解释都必须回到“帮用户观察自己和处境”，不能替代医疗、法律、财务、亲密关系等现实判断。

四维解析统一采用以下输出顺序：

1. `一句客观观察`：描述当前倾向，不下最终结论。
2. `可能卡点`：指出风险，但用可调整语言表达。
3. `现实提醒`：告诉用户需要回到现实验证什么。
4. `今日练习`：给一个低风险、可完成的小动作。
5. `边界提醒`：说明哪些情况不应依赖本产品。

### 1.3 四维解析维度定义

#### 事业

可说：

- 行动节奏、目标推进、沟通协作、压力来源、边界感。
- 用户可能是在“推进太急”“迟迟不动”“承担过多”“害怕冲突”中的哪一类。
- 传统根据可来自命宫气质、生日本命卦主题、流年宫节奏、五行生克关系。

不可说：

- 不预测升职、裁员、考试、创业成败。
- 不替用户决定辞职、跳槽、合作或重大职业选择。
- 不用“贵人必来”“小人作祟”等外归因话术。

推荐表达：

- “这更像是在提醒你：现在先把目标拆小，而不是继续硬扛。”
- “如果你已经连续失眠、效率明显下降，先处理压力负荷，再谈突破。”

#### 感情

可说：

- 亲密关系里的表达方式、依赖与边界、误会、期待落差、修复可能。
- 用户可能是在“想靠近但不敢说”“过度忍让”“控制感过强”“把旧情绪带入新关系”中的哪一类。
- 可以提醒用户回到事实、沟通和自我照顾。

不可说：

- 不断定对方爱不爱、是否出轨、是否复合、是否正缘。
- 不鼓励窥探、控制、测试、冷暴力或情绪勒索。
- 不把关系问题全部归因为命盘、卦象或某一方“命不好”。

推荐表达：

- “这不是在判断对方，而是在提醒你看见自己在关系里最容易重复的反应。”
- “先问清楚事实，再处理情绪；不要用猜测替代沟通。”

#### 身心

可说：

- 情绪消耗、睡眠节律、压力负荷、身体信号、自我照料。
- 用户可能是在“长期紧绷”“过度输出”“压抑感受”“缺少恢复”中的哪一类。
- 可以建议休息、记录、运动、减少刺激、寻求支持。

不可说：

- 不诊断疾病，不解释具体病因，不替代医生、心理咨询师或治疗方案。
- 不使用“必有病灾”“身体大凶”等恐吓话术。
- 不建议停药、换药、拒绝就医。

必须提醒：

- 如果严重或痛苦症状持续两周以上，或影响睡眠、饮食、起床、专注、日常任务，应建议寻求专业帮助。
- 如果用户表达自伤、自杀、无法保证安全，应提示立即联系当地紧急服务或危机热线；在美国可联系 988。

推荐表达：

- “这更像是一个压力负荷提醒，不是健康诊断。”
- “如果这种状态已经持续影响睡眠、饮食或日常功能，不要只靠自我解释，建议找专业人士聊一聊。”

#### 钱财

可说：

- 消费冲动、资源分配、安全感、储蓄节奏、风险意识、长期积累。
- 用户可能是在“靠消费安抚情绪”“过度保守”“短期收益焦虑”“人情支出失衡”中的哪一类。
- 可以提醒做预算、延迟决策、核对风险、保留现金流。

不可说：

- 不预测发财、破财、中奖、投资收益。
- 不推荐股票、基金、币、借贷、保险或任何具体金融产品。
- 不用“财星旺必发财”“此卦适合重仓”等诱导性表达。

必须提醒：

- 涉及投资、借贷、合同、担保、保险等现实财务决策，应咨询合格专业人士，并独立核实风险。
- 对“保证收益”“限时机会”“稳赚不赔”等说法保持警惕。

推荐表达：

- “这里提醒的是你的资源管理节奏，不是投资建议。”
- “如果一个选择让你必须立刻转钱、借钱或押上大额本金，先停下来核实。”

### 1.4 四维解析内容模板

每个维度固定输出：

- `这一维在提醒你`：一句白话判断。
- `你可能卡在`：一个可调整的现实卡点。
- `现实里先核对`：一个事实检查问题。
- `今天先做一步`：一个低风险行动。
- `传统根据`：说明来自命宫、卦象、流年、八星或五行的哪一层。
- `边界`：身心、钱财、感情等敏感领域必须有明确边界。

示例语气：

- 不说：“你今年感情必有劫。”
- 改说：“关系里容易出现拉扯，不一定是结果不好，而是你需要先分清事实、期待和情绪。”
- 不说：“你财运很旺，可以大胆投。”
- 改说：“资源流动感变强，适合整理预算和机会清单，不适合用冲动替代判断。”
- 不说：“你身体会出问题。”
- 改说：“这像是身心负荷提醒。先观察睡眠、饮食和精力，如果持续影响生活，应找专业支持。”

## 2. 当前项目已使用的术数体系

### 2.1 周易六十四卦

当前使用位置：

- 每日一卦
- 问易 / 竹签摇卦
- 数字起卦
- 占卦结果页
- 卦象图鉴
- 生日本命卦解释

当前数据内容：

- 六十四卦名
- 上卦 / 下卦结构
- 卦辞
- 象辞
- 爻辞与爻辞解释
- 关键词标签
- 阴阳爻图形展示

产品价值：

- 为用户提供“处境模型”，不是直接给答案。
- 适合解释情绪、关系、选择、行动节奏。
- 是观心最核心、最有文化根基的内容层。

不足：

- 卦辞、象辞、爻辞需要建立可追溯来源表，区分“原文”“白话解释”“产品化启发”。
- 图鉴页仍偏资料库，缺少“这个卦适合什么人生场景看”的入口。
- 结果页对“本卦 / 动爻 / 互卦 / 变卦”的层级仍可继续精炼。

优化建议：

- 每一卦统一拆成四层：`原文层`、`处境层`、`情绪层`、`行动层`。
- 每一卦补一个“人生场景标签”：关系、事业、选择、低谷、等待、转折、修复、边界。
- 每一卦补一句“现代白话钩子”，用于结果页首屏和分享卡片。

### 2.2 八卦与上下卦结构

当前使用位置：

- 六十四卦生成
- 卦象图鉴详情
- 生日本命卦计算
- 命宫 / 八宅 / 五行关系

当前数据内容：

- 乾、坤、坎、离、震、巽、艮、兑
- 天、地、水、火、雷、风、山、泽
- 五行、方位、家庭象、身体象、动物象等扩展属性

产品价值：

- 八卦是用户理解六十四卦的“底层拼图”。
- 上卦可解释外部环境，下卦可解释内在基础。
- 适合做新手教育和图鉴学习。

不足：

- 当前八卦属性展示偏百科，没有形成“如何用于看自己”的教学结构。
- 方位、五行、家庭象、身体象等内容容易过载，不适合一次全部展示。

优化建议：

- 新手层只讲八个自然象：天、地、水、火、雷、风、山、泽。
- 深读层再讲五行、方位、家庭象等传统对应。
- 结果页只展示对当前卦有用的上下卦关系，避免一次塞完整知识表。

### 2.3 梅花易数轻量起卦

当前使用位置：

- 以事问卦
- 竹签摇卦
- 数字起卦
- 占卦结果页

当前实现：

- 根据时间、问题或输入数字生成上卦、下卦、动爻。
- 生成本卦、互卦、变卦。
- 动爻用于解释当前变化点。

产品价值：

- 给用户一个“把当下问题外化”的仪式。
- 互卦和变卦天然适合做“现在卡在哪里 / 后续怎么变化”的叙事。

不足：

- 经典梅花易数常讲体卦、用卦、互卦、变卦及五行生克，当前产品还没有完整体现“体用关系”。
- 当前以毫秒时间戳起卦更像轻量仪式，应明确是产品化简化算法，避免冒充完整传统占法。
- 数字起卦在文档和界面之间存在差异：文档曾写两个数字，当前界面已有三个数字。

优化建议：

- 占卦结果页新增体用解释：`我方状态`、`外部问题`、`中间结构`、`变化方向`。
- 将互卦解释成“问题内部结构”，变卦解释成“若继续这样走的趋势”。
- 对外文案避免“正统梅花全法”，改为“参考梅花易数思路的轻量起卦”。

### 2.4 命宫 / 命卦 / 生日本命卦

当前使用位置：

- 生辰信息主路径
- 命理档案页
- 命宫解读页
- 八宅八星自我镜

当前实现：

- 根据出生年份和性别计算命宫 / 命卦。
- 根据年、月、日、时组合计算一个六十四卦，当前页面多处称为“本命卦”。
- 根据命宫和当前年份计算流年宫。

产品价值：

- 这是当前最能做“个性化”的模块。
- 用户输入生日后，天然期待“和我有关”的内容。
- 可承接情绪按钮、长期主题、今年节奏、今日练习。

不足：

- “本命卦”容易与八宅体系里的“命卦”混淆。
- 当前不是完整八字系统，但部分文档和页面仍写“八字”“排四柱”，需要修正。
- 生日本命卦当前已明确采用“生辰四柱天干 -> 纳甲数 -> 天数 / 地数 -> 上下卦 -> 六十四卦”的计算口径。
- 这一层用于观察长期反复出现的处事主题，不等同于八宅命卦，也不是完整八字十神排盘或紫微斗数命盘。

术语统一建议：

- `命宫 / 命卦`：个人方位与性格底色观察，用于八宅相关内容。
- `生日本命卦`：观心用生辰组合出的六十四卦观察层，不等同于八宅命卦。
- `流年宫`：今年节奏参考，不做命运预测。
- `生辰信息`：当前表单输入项，不称“八字”，除非后续真实加入年柱、月柱、日柱、时柱及节气换月。

内容结构建议：

- 第一层：我为什么会这样反应。
- 第二层：我容易卡在哪里。
- 第三层：今天怎么练。
- 第四层：如果想学习传统文化，再看术语和出处。

### 2.5 八宅八星

当前使用位置：

- 命宫解读页“八宅八星自我镜”
- 八宅风水相关计算

当前数据内容：

- 东四命 / 西四命
- 八方位
- 八宅游年星：生气、天医、延年、伏位、绝命、五鬼、六煞、祸害
- 五行与吉凶分类

产品价值：

- 可以为命宫提供“多维了解自己”的补充。
- 适合拆成“四个助力 / 四个提醒”，不做恐吓式吉凶。
- 能让用户在使用中学到八宅文化，但不必进入风水摆件和空间改造。
- 更适合作为“八星性格镜 / 八星磁场观察”的副层模块，而不是产品的主底层术数。

不足：

- 八宅本来偏方位与宅法，放到自我理解产品里必须解释“这是文化观察，不是空间改造建议”。
- 游年星的来源和算法需要继续校验，尤其是星曜与方位的对应表。
- “绝命、五鬼”等词容易制造恐惧，需要持续弱化为风险管理语言。

方向判断：

- 八宅八星的原始语境仍是阳宅方位系统，因此不适合继续承担观心的主产品内核。
- 观心里保留八星的理由，不是把它做成完整风水功能，而是借其“星性 + 数字 + 方位”的结构，作为用户观察性格倾向、关系磁场、近期状态的辅助镜面。
- 后续如继续保留，产品命名应逐步从“八宅风水”收口为“八星性格镜”或“八星磁场观察”，并在说明中明确：`借八宅八星做自我观察，不做阳宅风水判断。`

术语转译建议：

- 生气：成长动能、主动打开。
- 天医：修复滋养、身心照料。
- 延年：关系稳定、长期协作。
- 伏位：沉淀守成、回到原点。
- 绝命：高压高消耗提醒。
- 五鬼：扰动、误会、内耗。
- 六煞：牵连、情绪拉扯。
- 祸害：小摩擦、小失误。

优化建议：

- 保留“助力 / 提醒”的表达，不直接展示“大凶”等吓人标签。
- 每颗星固定四层解释：传统含义、生活场景、今天怎么用、不要怎么误解。
- 后续可以增加“我的助力方向”和“我的提醒方向”，但不建议指导用户摆放物品。

### 2.6 五行生克

当前使用位置：

- 命宫与流年五行关系
- 生日本命卦上下卦五行
- 八宅八星五行

当前关系：

- 生我
- 我生
- 我克
- 克我
- 比和

产品价值：

五行关系很适合转译成现代心理语言：

- 生我：被滋养、可借力。
- 我生：输出、消耗、表达。
- 我克：掌控、推进、处理。
- 克我：压力、约束、修炼。
- 比和：共振、强化、稳定。

不足：

- 当前五行解释仍有一些“贵人、机遇、压力”等泛化表达，容易说了像没说。
- 没有与用户当前具体场景绑定。

优化建议：

- 五行关系必须落到三个生活问题：我该借力、该收力，还是该推进？
- 与历史记录结合，判断用户近期是“持续输出过大”“压力过强”还是“反复停在舒适区”。
- 五行只作为节奏语言，不作为命运判断。

### 2.7 节气与时间文化

当前使用位置：

- 项目中已有节气数据雏形和后续规划。
- 当前尚未形成完整节气专题体验。

产品价值：

- 节气很适合观心的产品定位，因为它天然连接“天地节律”和“个人生活节奏”。
- 比复杂术数更容易被普通用户接受。
- 可用于日常留存：每个节气一个主题、一个提醒、一个练习。

不足：

- 当前节气计算是基于固定参考点的近似，不能直接作为长期准确历法能力。
- 还没有将节气内容接入每日练习、问卦或命宫回看。

优化建议：

- 第一阶段不做复杂历法，只做“节气主题内容”。
- 每个节气固定四层：物候、情绪主题、修身提醒、今日小练习。
- 后续如做精确节气时间，需接入可靠历法数据或算法库。

## 3. 需要立即修正的表达问题

### 3.1 文档和页面中过度承诺“八字”

当前代码并未真正计算天干地支四柱，因此不应在产品层写“八字计算”“排四柱”。

短期统一改为：

- `生辰信息`
- `命宫`
- `生日本命卦`
- `流年宫`

除非后续加入真实四柱能力，否则不要使用“八字排盘”“四柱排盘”作为功能承诺。

### 3.2 “本命卦”命名需要更清楚

“本命卦”作为用户容易理解的短名可以保留，但页面首次出现时建议写成：

**生日本命卦**

说明文案：

> 这是观心用你的生辰信息组合出的六十四卦观察层，用来看长期主题；它不等同于八宅里的命卦，也不是完整八字。

### 3.3 吉凶词需要产品化转译

传统文本里的吉、凶、悔、吝可以保留在学习层，但用户首屏不应直接被“凶、大凶、绝命”等词刺激。

建议转译：

- 吉 -> 助力
- 凶 -> 提醒
- 悔 -> 复盘
- 吝 -> 卡点
- 绝命 -> 高压 / 高消耗提醒
- 五鬼 -> 扰动 / 误会 / 内耗
- 六煞 -> 牵连 / 情绪拉扯
- 祸害 -> 小摩擦 / 小失误

## 4. 推荐补充的传统文化哲思层

### 4.1 易有三义：变易、简易、不易

适合做观心的总哲学：

- `变易`：情绪和处境都在变化，不要把当下当永恒。
- `简易`：复杂问题先抓一个最小行动。
- `不易`：变化中仍有原则和边界。

产品落点：

- 结果页可增加“此刻在变什么 / 你能先抓什么 / 什么原则不要丢”。

### 4.2 君子以修身为核心

《易传》大量“君子以……”句式，适合观心做传统文化学习和行动建议。

产品落点：

- 每个卦的象辞不要只翻译，要转成一个“君子练习”。
- 例如乾卦不是“你会成功”，而是“今天练自强但不逞强”。

### 4.3 观象，不是迷信结果

八卦核心是“取象”：用自然图像帮助人理解状态。

产品落点：

- 所有术数都先回答：这个象在提醒用户看见什么？
- 不先问“准不准”，先问“它帮我看见了什么？”

## 5. 内容输出统一模板

后续所有术数解释统一使用以下模板：

1. `一句白话`：先告诉用户这在说什么。
2. `对应你`：它和用户当前情绪、关系、选择有什么关系。
3. `传统根据`：来自卦象、卦辞、象辞、爻位、八宅星或五行关系的哪一层。
4. `今天怎么用`：一个当天可完成的小动作。
5. `不要误解`：不做宿命论、不替代现实判断、不制造恐惧。

### 5.1 术数表达三层法

观心的文字不能只有“温柔安慰”，也不能只有“术数断语”。推荐统一采用三层表达：

1. `术数骨架`：保留卦名、象、爻、五行、八星、命宫等传统依据，明确“为什么这样说”。
2. `现实翻译`：把术数含义翻译成用户当下能理解的处境、关系、情绪和行动问题。
3. `情绪承接`：指出可调整之处，给出一个不恐吓、不粉饰的下一步。

这三层的关系是：

- 没有术数骨架，内容会像普通心理鸡汤，缺少传统文化魅力和准确感。
- 没有现实翻译，内容会像术语堆砌，零基础用户无法进入。
- 没有情绪承接，内容会显得生硬，容易放大焦虑或制造宿命感。

### 5.2 文案强度控制

每段解读都要控制在“有锋芒，但不伤人”的强度：

- `术数原意要保留`：例如“坎为险”“绝命为高压高消耗”“克我为压力约束”，不能全部改成空泛的“最近要照顾自己”。
- `断语要降级为倾向`：例如不说“你会失败”，改为“这类结构容易让你在推进中遇到阻力，需要先看清阻力来自哪里”。
- `安慰不能抹平问题`：例如不说“一切都会好”，改为“这不是坏结果的判定，但它提醒你现在不能只靠忍耐拖过去”。
- `行动必须具体`：每段至少给一个当天能做的小动作，不只给价值观口号。

推荐句式：

- `传统上这叫……，它不是在说……，而是在提醒你……`
- `这个象的锋芒在于……，落到现实里更像……`
- `如果你现在确实处在……，先不要急着下结论，先核对……`
- `今天先做一件小事：……`

### 5.3 示例：从断语到观心表达

示例一：坎象

- 过直表达：“坎为险，你最近会遇到危险。”
- 过度修饰：“最近有些不确定，照顾好自己就好。”
- 观心表达：“坎的本意是险，是水中有陷。它不等于坏事会发生，而是在提醒你：现在的信息可能不够清楚，情绪也容易把问题放大。先别急着定结论，今天先把最担心的一件事写成事实、猜测、可行动三栏。”

示例二：五鬼星

- 过直表达：“五鬼主是非，小心小人。”
- 过度修饰：“最近沟通上可能有点小波动。”
- 观心表达：“五鬼在八宅里是扰动之星，传统上常指误会、暗耗和杂念。放到现实里，它更像提醒你：别让猜测替代沟通。今天先确认一个事实，不要急着给别人动机下判断。”

示例三：财务主题

- 过直表达：“财运不稳，容易破财。”
- 过度修饰：“最近花钱上可以更谨慎一点。”
- 观心表达：“这个结构显示资源流动感变强，不是说你一定破财，而是提醒你容易用消费缓解不安。今天先做一件事：把本周非必要支出列出来，任何大额决定延迟 24 小时再做。”

### 5.4 内容验收标准

后续所有命宫、四维解析、问卦 AI 解读、图鉴白话都按以下标准验收：

- 是否保留了至少一个明确的传统依据。
- 是否把传统依据翻译成了现实处境，而不是只换成抽象形容词。
- 是否指出了真实卡点，而不是只给温柔安慰。
- 是否避免了“必然、注定、大凶、稳赚、正缘、必分”等确定性断言。
- 是否给了一个具体、低风险、当天可做的行动。
- 是否在身心、钱财、感情等敏感主题上给出边界提醒。

## 6. 优先级建议

### P0：先做准确性收口

- 修正文档和页面中“八字 / 排四柱”的表述。
- 建立术语说明：命宫、命卦、生日本命卦、流年宫、八宅八星。
- 给六十四卦原文和解释做来源标注。
- 将术数总链路统一为：`八卦为体，梅花为用，命宫 / 生日本命卦为人，五行为翻译层，节气为留存层`。

### P1：提升个性化击中感

- 命宫页继续压缩重复文案，强化“情绪按钮 / 卡点 / 今日练习”。
- 命宫页新增“四维生活解析”：事业、感情、身心、钱财。每一维只输出一句判断、一个卡点、一个现实核对问题、一个今日练习、一个传统根据和一个边界提醒。
- 占卦结果页增加体用关系，用“我方状态 / 外部问题”解释。
- 历史记录按主题聚合，反推用户近期反复出现的卦象主题。
- 八星模块降级为副层：保留其学习价值和互动趣味，但不再让它承担产品底层解释框架。

### P2：增加学习与留存

- 每日一卦升级为“每日一象”：一句象辞哲思 + 一个今日练习。
- 节气专题做成 24 个轻量内容节点。
- 图鉴增加“人生场景索引”，让用户按问题而不是按卦名学习。
