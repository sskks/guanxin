# 观心 (Guān Xīn)

以易观心 · 明悟自性

一款基于周易文化的单页 Web 应用，融合梅花易数占卦、命宫解读、六十四卦图鉴等功能，以古朴水墨风格呈现传统易学智慧。

## 功能

- **每日一卦**：六十四卦逐日品读，含卦象图解与白话解读
- **以事问卦**：心中所惑，摇卦求指引，支持 AI 解读（通义千问）
- **命宫解读**：基于出生信息的个人卦象特质与流年五行分析
- **卦象图鉴**：六十四卦百科，支持搜索与八卦筛选
- **数字起卦**：随心报数，以数取卦
- **占卦历史**：浏览过往占卦记录，支持收藏与搜索
- **每日一卦打卡**：连续打卡记录，养成学习习惯

## 技术

- 纯前端单文件 SPA（HTML/CSS/JS 全内联，无框架依赖）
- IndexedDB + localStorage 本地持久化
- DashScope API（通义千问 qwen-plus）AI 解读
- 部署于 Cloudflare Pages
